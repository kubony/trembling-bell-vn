# 흔들리는 종소리 (Earthquake Drill VN)

교실에서 지진이 발생한 순간, 히로인 **이수민**과 함께 안전하게 대피하는 과정을 다룬 교육용 비주얼노벨 웹 게임입니다. 선택에 따라 **안전 / 호감 / 침착** 세 가지 지표가 변하며, 올바른 지진 대피 요령을 자연스럽게 익힐 수 있습니다.

## 플레이 방법

별도의 빌드 과정 없이 정적 파일만으로 동작합니다.

```bash
# 저장소 클론 후 로컬 서버 실행 (예시)
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속 후 플레이하세요. (`index.html`을 직접 열어도 대부분 동작합니다.)

## 구성

- `index.html` — 게임 화면 구조
- `app.js` — 스토리 진행 및 선택지/지표 로직
- `styles.css` — 비주얼노벨 UI 스타일 및 지진 연출
- `assets/` — 배경·캐릭터 스프라이트 (`generated/`는 게임에 사용되는 렌더 이미지)

## 지표

- **안전** — 올바른 대피 행동으로 상승
- **호감** — 이수민과의 상호작용
- **침착** — 침착하게 상황에 대응

---

지진 대피 교육을 게임 형태로 익히는 것을 목표로 제작되었습니다.
